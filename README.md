# smart-learning-analytics-backend

基于大模型的学生学情分析与预警系统

项目经历：我曾参与“智能学情分析后台”项目，负责通过大模型技术优化学生行为数据分析模块。我协助产品经理梳理了教师对学情预警的需求，并与研发合作接入了LLM接口，实现对学习轨迹的自动归类与风险标注。项目上线后，系统对潜在学困生的识别准确率提升了18%，后台周均使用频次增加了25%。我还通过SQL定期分析功能使用数据，协同测试团队优化了模型响应速度与报表生成的稳定性。

## 运行
```bash
pip install -r requirements.txt
python main.py
```
